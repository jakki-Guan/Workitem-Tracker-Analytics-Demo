# Security & Privacy

## What is excluded
- No tenant URLs, list GUIDs, workspace names, or internal identifiers
- No user names/emails
- No record-level descriptions, notes, or vendor references
- No production datasets

## Redaction rules (Screenshots)
- Remove all org branding
- Replace names with pseudonyms (Lead A..N)
- Replace categories with Workstream 1..N
- Hide any table/detail visuals (record list)
- Remove timestamps that could identify operations

## Naming policy
All public artifacts use demo naming:
WorkItemKey / Lead / Workstream / Stage / UrgencyBand / SLAFlag
